# Iris Species Classification using Machine Learning

## Overview
This project implements machine learning algorithms to classify iris species based on their physical characteristics.

## Dataset
The Iris dataset contains 150 samples of iris flowers with 4 features:
- Sepal length
- Sepal width
- Petal length
- Petal width

The dataset includes 3 iris species:
- Setosa
- Versicolor
- Virginica
